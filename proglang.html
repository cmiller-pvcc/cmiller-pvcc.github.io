Chapter 6: Programming Languages

TERM Definition
Machine Language uses the binary code for the computers processor
Debugging identifies and removes errors in a program
Assembler transforms low-level assembly code by human into machine code that can be executed by a computer's processor.
Assembly Language programming language that is specific to a particular computer
Translator converts code written in one programming language into an code written in another programming language
compiler translates advanced code into machine-readable code that can be executed by a computer
interpreter helps a computer understand and execute instructions written in a programming language
cross-platform software programs that can run on multiple types of devices and operating systems
natural language how computers use our native language to make it easier to understand
formal language a set of rules that define how code should be written
machine independence where the program can be used on different hardware and software configurations
high-level programming language allows humans to work with computers by writing code that is closer to a natural languag
object-oriented programming (OOP) organizes software design around data, or objects, rather than functions
scripting language used to execute specific tasks and designed to make it easier
integer (Data Type) a variable that contains whole numbers
float (Data Type) a variable that contains decimal numbers
string (Data Type) a variable that contains characters. includes letters, numbers, and symbols
primitive data type basic building blocks of a programming language. can't be broken down into smaller pieces.
data structure way to organize or store data in a program
python collection data structures (pycds)
list (pycds) allows you to store a sequence of values or items
tuple (pycds) allows you to group together multiple values into a single text
set (pycds) stores unique elements of the same type in a sorted order
dictionary (pycds) stores and acceses data in a way that's easy to understand and efficient to search
function takes input, processes it, and produces output
local variable data that can only be accessed within a specific part of code
global variable a variable that can be accessed from anywhere in the program
source program a set of instructions written in the program language that a computer can understand
object program a file created by a compiler that contains the machine code or executable code that can be run on a computer
