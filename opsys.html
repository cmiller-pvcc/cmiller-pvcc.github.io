Operating Systems Terminology

TERM Definition
operating System a software that manages the computer hardware resources and gives common services for computer programs
job queue a list of various tasks waiting to be done by the operating system
FIFO Stands for the phrase "first-in,first-out" which means that the first item that gets added to the queue will also be the first item to be removed and proccessed.
interactive processing where the usere communicates with the computer
time-sharing lets multiple users access the computer and its resources at the same time
multiprogramming technique which allows many different programs to be used at the same time by one processor
application software a specific type of computer software that only does certain tasks for the user
system software a type of software that manages/controls the hardware and also provides services to the application software
utility software a software that enhances or extends the functionality of an operating system
graphical user interface (GUI) an interface that that lets users interact with the software with things like icons or menus
shell command-line interface that allows users to interact with the operating system by typing in commands
kernel manages system resources and also provides services to the other programs
device driver allows the operating system to communicate with a hardware device
file manager provides a graphical user interface for sorting different computer files and folders
memory manager responsible for managing memory resources in a computer system
virtual memory memory management way that allows programs to execute which makes them think that have more memory then they actually do
boot strapping loads and starts up the operating system of a computer when its turned on
ROM stands for Read-Only Memory. remembers content wven when the power is turned off.
deadlock a problem where two or more processes are not able to continue because it is waiting for a resource from another
spooling stops storing data temporarily to optimize data transfer
super user (or administrator) a priviledged account with access to all settings
multi-factor authentication security method that makes users provide two or more authentification factors to access the system
